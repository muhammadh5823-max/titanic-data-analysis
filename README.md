# Titanic Data Analysis

This project analyzes the famous Titanic dataset using Python and Pandas.

## Project Goals

- Explore the dataset
- Understand data structure and columns
- Handle missing values
- Analyze passenger demographics
- Calculate survival statistics
- Compare survival rates by gender and passenger class

## Dataset Information

- Total Passengers: 891
- Features: 12 columns
- Missing values handled in:
  - Age
  - Embarked
- Cabin column removed due to excessive missing values

## Analysis Performed

### Basic Dataset Exploration
- Dataset shape
- Column names
- Data types
- Missing values

### Data Cleaning
- Filled missing Age values using mean
- Filled missing Embarked values using mode
- Dropped Cabin column

### Statistics
- Total passengers
- Male/Female count
- Average age
- Highest fare
- Total survivors
- Survival rate

### Survival Analysis
- Survival rate by Gender
- Survival rate by Passenger Class

## Technologies Used

- Python
- Pandas

## Key Findings

- Overall survival rate: 38.38%
- Female passengers had a much higher survival rate than males.
- First-class passengers had the highest survival rate.
- Third-class passengers had the lowest survival rate.
